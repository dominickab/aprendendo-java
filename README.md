# Aprendendo Java

Este repositório foi criado para registrar meu processo de aprendizado em Java.  
Aqui estão exercícios simples feitos com o objetivo de praticar a lógica, a sintaxe da linguagem e a organização de um projeto Java.

Os códigos foram desenvolvidos conforme avanço nos estudos, começando pelos conceitos mais básicos.

## Conteúdos praticados até o momento

- Estrutura básica de um programa em Java
- Método main
- Organização de pacotes
- Entrada e saída de dados
- Estruturas condicionais (if / else)
- Estruturas de repetição (while)
- Exercícios práticos para fixação da lógica
