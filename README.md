# Aprendendo Java

Este repositório foi criado com o objetivo de estudar e praticar a linguagem Java, iniciando pelos conceitos fundamentais e evoluindo de forma progressiva por meio de exercícios práticos.

## Conteúdos estudados até o momento

- Estrutura básica de um programa em Java
- Método main
- Organização de pacotes
- Entrada e saída de dados
- Estruturas condicionais (if e else)
- Estruturas de repetição (while)
- Exercícios práticos para fixação da lógica

Consolidar os fundamentos da linguagem Java, criando uma base sólida para o avanço em programação orientada a objetos e desenvolvimento backend.
